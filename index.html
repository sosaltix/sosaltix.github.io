Проект sosaltix2

Маленькая операционная система на Rust, реализующая базовое ядро x86_64 с поддержкой прерываний, управления памятью и асинхронной кооперативной многозадачности.

Ключевые Особенности

Функционал ядра sosaltix2, реализованный полностью на низкоуровневом стеке.

Стек Rust (no_std)

Безопасность работы с памятью прямо на уровне "железа" без использования стандартной библиотеки Rust.

Кооперативный Async

Полноценная поддержка конструкций async/await и легковесная кооперативная многозадачность.

Поддержка FAT32

Интегрированная работа с файловой системой FAT32: создание, чтение, изменение и удаление файлов.

Пейджинг и куча

Реализована виртуальная память с страничной адресацией (paging) и кастомная динамическая память (heap allocator).

Драйвер клавиатуры

Низкоуровневая обработка прерываний и корректный парсинг скан-кодов с PS/2 клавиатуры.

Запуск в QEMU

Ядро полностью совместимо со спецификацией архитектуры x86_64 и готово к отладке в эмуляторах.

Квота на Rust (50%+)

Любое модифицированное произведение обязано содержать не менее 50% исходного кода на Rust. Объем рассчитывается утилитой cloc (>=1.96) или tokei (>=12.1.2).

Запрет на Unsafe (≤1.0%)

Строгий математический лимит: объем кода внутри контекста unsafe не может превышать 1.0% от общего объема кода на Rust. Контролируется через cargo-geiger (>=0.11.0).

Сборка и Запуск ядра

Пошаговое развертывание среды разработки под nightly-ветку Rust.

1

Переключение на Nightly

Проект активно задействует нестабильные nightly-фичи ядра компилятора.

2

Установка системных компонентов

Нам понадобятся исходники rust-src и тулчейн llvm-tools-preview для линковки bare-metal.

3

Запуск UEFI

Загрузка и компиляция готового бинарного образа в виртуальной машине QEMU.

bash (Arch Linux)

# 1. Активируем nightly-профиль

rustup default nightly

# 2. Добавляем компоненты для bare-metal сборки

rustup component add rust-src

rustup component add llvm-tools-preview

# 3. Клонируем проект и собираем под UEFI

git clone https://github.com/ktoto42-oss/sosaltix2.git

cd sosaltix2

cargo run uefi
